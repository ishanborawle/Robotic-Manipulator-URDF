# Robotic-Manipulator

## 📌 Overview

This repository contains a **3-DOF Robotic Manipulator**.

The project includes the robot description, individual **STL (.stl) mesh files**, and a display launch file for robot model visualization.

---

## 🎯 Objectives

* Create a 3-DOF robotic manipulator model.
* Export individual robot links as STL mesh files.
* Create a Xacro/URDF robot description.
* Define robot links and joints.
* Integrate CAD meshes into the robot model.
* Understand robot description and visualization.
